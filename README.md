# GoodHamburger 🍔

Sistema de gestão de hamburgueria desenvolvido com **.NET 9/10**.

## 🚀 Tecnologias
* ASP.NET Core Web API
* Entity Framework Core
* SQLite (Banco de dados local)

## 🏗️ Estrutura do Projeto
* **Domain**: Entidades de negócio (ItemCardapio, Pedido).
* **Application**: Regras de aplicação.
* **Infrastructure**: Acesso a dados e persistência (SQLite).
* **Api**: Porta de entrada do sistema e Controllers.
